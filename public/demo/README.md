# Demo Map Asset

Place the real built-in satellite preview image here:

`public/demo/default-map-preview.png`

The image should be a Google Static Maps satellite snapshot for the default source settings:

- latitude: `43.6554642`
- longitude: `-79.5870194`
- zoom: `16`
- map labels: off (`maptype=satellite`)
- standard source size: `640x640`, `scale=2` (`1280x1280` pixels)

Do not use generated placeholder art here. The application uses this file only when no Google API key is configured.
